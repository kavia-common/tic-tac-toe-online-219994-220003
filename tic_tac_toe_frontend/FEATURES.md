# Tic Tac Toe - Ocean Professional

Features:
- Playable 3x3 board with click/tap interactions
- Modes: Player vs Player, Player vs AI (basic win/block heuristic)
- Game state: turn tracking, win/draw detection
- Score tracking (session lifetime): X wins, O wins, draws
- Controls: mode toggle, Reset round, New Match, and Play Again on result
- Responsive, centered layout with modern Ocean Professional styling

Run:
- npm install
- npm run dev
- Open http://localhost:3000
